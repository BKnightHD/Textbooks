## 🎯 Purpose

This repo exists purely for personal self-study and reference. The goal is to have a curated reading list covering:

- **Finance & Investing** — securities, corporate finance, portfolio theory
- **Mathematics** — probability and statistical foundations
- **Computer Science** — algorithms and data structures
- **Data & Programming** — data engineering pipelines, Python for analysis
